<h1>Description</h1>
<p>A simple side-scrolling running game built with HTML, CSS, and JavaScript.</p>

<h1>Goal</h1>
<p>Collect coins to increase your score and avoid hitting stones.</p>

<h1>Controls</h1>
<p>Press the SPACEBAR to make Mario jump.</p>

<h1>Gameplay</h1>
<p>Background and ground scroll continuously to simulate motion.</p>

<h1>Coins</h1>
<p>Each coin collected adds +10 points to your score.</p>

<h1>Restart:</h1>
<p>After “Game Over,” the game automatically reloads.</p>
